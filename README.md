# Anime In Terminal

Watch anime in your terminal in questionable quality.
Choose from two different renderers.



## Installation

Install dependencies:

```bash
apt install fzf mpv caca-utils 
```

## Usage

```bash
cd animeInTerminal
```

```bash

./anime.sh 
```
